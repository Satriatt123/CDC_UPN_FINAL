# CDC UPN FINAL

Fullstack project (Frontend + Backend + MySQL)
Developed by Dzinggo & Team

A. Download & Install Git
Buka https://git-scm.com/downloads
Pilih Windows, lalu install dengan semua pengaturan default (klik Next terus).
Setelah selesai, buka Command Prompt (CMD) atau PowerShell dan ketik: git --version

B. Download & Install Node.js
Buka https://nodejs.org/en/
Pilih versi LTS (Long Term Support).
Setelah selesai install, cek dengan:
node -v
npm -v

C. Download Project dari GitHub
Buka terminal (CMD atau PowerShell).
Masuk ke folder tempat kamu ingin menyimpan project: cd folder terserah
Clone project: git clone https://github.com/Satriatt123/CDC_UPN_FINAL.git
Masuk ke folder project: cd CDC_UPN_FINAL

D. Instalasi Dependensi
Frontend (React / Vite)
Masih di folder CDC_UPN_FINAL, jalankan: npm install

Backend (Node.js + Express)
Masuk ke folder backend: cd backend
Masih di folder backend, jalankan: npm installnpm install

E. Jalankan Backend
Di folder backend, jalankan: npm run dev
