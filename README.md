# CDC UPN FINAL

Fullstack project (Frontend + Backend + MySQL)
Developed by Dzinggo & Team

## Cara Menjalankan di Lokal

### Backend
1. Masuk ke folder `backend`
2. Jalankan:
   buka terminal, tulis cd + path folder backend
   npm install
   npm run dev

### Frontend
1. Masuk ke folder root
2. Jalankan:
buka terminal, tulis cd + path root
npm install
npm run dev
